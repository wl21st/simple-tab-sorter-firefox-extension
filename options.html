📋 Tab Manager Settings

Google Drive Integration
Connect your Google account to save tabs directly to Google Drive. Your files are saved to a "Tab Manager" folder in your Google Drive.
⚠️ Not connected to Google Drive
ℹ️ Token Guide:
  • Access Token: Required - grants permission to Google Drive
  • Refresh Token: Optional - enables automatic token refresh (recommended)
  • Token Expiry: Access tokens last ~1 hour. Refresh tokens work indefinitely (if you have a refresh token, auto-renewal is enabled).
⏰ Option 1: Quick Setup (Temporary - 1 hour)

Use this for quick testing. Tokens expire after ~1 hour and must be manually refreshed.

  1. Visit Google OAuth2 Playground →
  2. In the right panel (OAuth 2.0 Configuration), click the gear icon ⚙️ and check "Use your own OAuth credentials"
  3. On the left panel, search for "Drive API v3"
  4. Select scope: https://www.googleapis.com/auth/drive
  5. Click "Authorize APIs" and log in with your Google account
  6. Click "Exchange authorization code for tokens"
  7. Copy the access_token value from the response and paste above
  8. Optionally, copy the refresh_token and paste above for auto-refresh
  9. Click "Save & Test Token" to validate
⚠️ Refresh when expired: Use the "🔄 Retest Token" button to validate. If expired, repeat steps 5-7 to get a new access token.
✅ Option 2: Recommended Setup (Permanent - Auto-refresh)

One-time setup that works indefinitely. Tokens refresh automatically in the background.

  1. Go to Google Cloud Console →
  2. Create a new project named "Tab Manager"
  3. Go to APIs & Services → Enable "Google Drive API"
  4. Go to Credentials → Click "Create Credentials" → OAuth 2.0 Client ID
  5. Select "Chrome Extension" as the application type
  6. Add your Extension ID (shown in the "Extension Information" section below)
  7. Copy the generated Client ID and Client Secret
  8. Create a file named oauth-config.json in the extension folder:
{
  "clientId": "YOUR_CLIENT_ID.apps.googleusercontent.com",
  "clientSecret": "YOUR_CLIENT_SECRET"
}
  1. Reload the extension (go to chrome://extensions)
  2. Click the extension's "Authorize" button in the popup
  3. Tokens will auto-refresh automatically - no manual intervention needed!
💚 Auto-refresh: Tokens refresh automatically. Use "🔄 Retest Token" to check status anytime.
🔄 How to Manually Refresh Your Token:

If your token expires and you need to refresh it manually:

  1. If you have a Refresh Token: Paste your refresh token in the field above. The extension will automatically refresh the access token when needed.
  2. If you only have an Access Token: Go back to OAuth2 Playground and repeat steps 5-7 to get a new access token, then paste it above.
  3. Check Status: Click "🔄 Retest Token" to verify your token is valid.
💡 Tip: Keep your refresh token safe! If you have one, you won't need to manually refresh for a very long time.
