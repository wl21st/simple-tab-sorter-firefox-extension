## Basic Information

### Extension Name
Simple Tab Manager

### Short Description (80 characters max)
Sort, merge, and extract tabs by domain efficiently. Organize your browser tabs.

### Full Description
Simple Tab Manager is a lightweight and efficient Chrome extension designed to help you organize and manage your browser tabs with ease.

**Key Features:**
- **Sort Tabs** - Alphabetically sort all tabs by URL
- **Merge All Windows** - Combine multiple windows into one
- **Extract Same Domain** - Instantly extract all tabs from the same domain across all windows into a new window
- **Remove Duplicates** - Eliminate duplicate tabs by URL
- **Auto-Pause Videos** - Automatically pause videos on Bilibili and YouTube when managing tabs

Perfect for users who work with many tabs and need quick organization tools.

## Permissions Justification

### Why each permission is needed:

- **tabs** - Required to query, identify, and move tabs between windows
- **windows** - Required to create new windows when extracting tabs by domain
- **scripting** - Required to pause video playback on supported sites (Bilibili, YouTube)

We do NOT collect any user data. All operations are local to your browser.
